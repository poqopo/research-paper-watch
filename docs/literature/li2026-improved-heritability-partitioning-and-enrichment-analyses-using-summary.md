# Improved heritability partitioning and enrichment analyses using summary statistics with graphREML

## Healthcare relevance

Improves prioritization of tissues, annotations, and mechanisms underlying genetic risk for complex diseases.

## Analysis

- Abstract: graphREML is a likelihood-based heritability partitioning method that uses genome-wide association studies summary statistics and linkage disequilibrium graphical models. It yields enrichment estimates concordant with S-LDSC but shows greater power and produces well-calibrated estimates of per-SNP heritability.
- Prior limitation: Common enrichment tools such as S-LDSC can be limited by power and calibration when estimating per-SNP heritability from summary statistics.
- Differentiator: Introduces graphREML, a likelihood-based method using LD graphical models for calibrated and more powerful heritability partitioning.
