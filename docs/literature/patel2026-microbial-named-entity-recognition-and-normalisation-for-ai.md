# Microbial Named Entity Recognition and Normalisation for AI-assisted Literature Review and Meta-Analysis

## Healthcare relevance

Supports microbiome and infectious-disease literature curation for meta-analysis.

## Analysis

- Abstract: MOTIVATION: Manual curation of biomedical literature is slow and error-prone and while large language models trained on general texts have shown to be useful for text summarisation, these methods lack the domain-specific expertise required to perform this task accurately.  Here we describe the creation of the first microbiome-specific text corpus, use this to train deep learning algorithms for named-entity recognition (NER) and entity linking (EL), and demonstrate their use to meta-analyse microbiome literature.  RESULTS: The training and validation set (n = 1,410) contained a total of 90,150 annotations (both long form and abbreviations).
- Prior limitation: Manual curation of biomedical literature is slow and error-prone and while large language models trained on general texts have shown to be useful for text summarisation, these methods lack the domain-specific expertise required to perform this task accurately.  Here we describe the creation of the first microbiome-specific text corpus, use this to train deep learning algorithms for named-entity recognition (NER) and entity linking (EL), and demonstrate their use to meta-analyse microbiome literature.
- Differentiator: The training and validation set (n = 1,410) contained a total of 90,150 annotations (both long form and abbreviations).  Using the gold-standard test set (n = 288), with an inter-annotator agreement rate of 99.
