# A one-week automated genome-wide optical pooled screen using OttoSeq.

## Healthcare relevance

Improves scalable functional annotation of genes and perturbations, which can support target discovery and disease mechanism screening.

## Analysis

- Abstract: Optical pooled screens (OPS) are bottlenecked by labor-intensive in situ sequencing and analysis protocols.  Here, we present OttoSeq, an automated OPS platform combining the Otto2 fluid handling system with the Brieflow analysis pipeline.  We utilize OttoSeq to complete a genome-wide cell painting screen in eight days, sampling 5,198,240 high-quality cells across 21,732 gene knockout perturbations (224 cells per gene) and interpreting 320 functional gene clusters.
- Prior limitation: Optical pooled screens were limited by labor-intensive in situ sequencing and analysis workflows.
- Differentiator: Combines automation and analysis into an eight-day genome-wide OPS workflow with millions of cells and thousands of perturbations.
